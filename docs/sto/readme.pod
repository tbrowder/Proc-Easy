=begin pod


=head1 NAME

B<Proc::Easy> - Provides routine C<run-command> to ease using Raku's B<Proc> class.  

=SYNOPSIS

=begin code
    use Proc::Easy;
    my $cmd = "some-user-prog arg1 arg2";
    my $other-dir = $*TMPDIR";
    my ($exitcode, $stderr, $stdout) = run-command $cmd, :dir($other-dir), :all;
=end code

=head1 DESCRIPTION

The routine is described in detail in
[ALL-SUBS](https://github.com/tbrowder/Proc-Easy/blob/master/docs/ALL-SUBS.md)
which shows a short description of the exported routine along with
its complete signature.
